# Fate Twin

This repository accompanies the manuscript:

**Fate Twin: Simulating Formation Processes for Understanding Ancient Manuscript Fragmentation**

The repository is intended to provide code, derived simulation metadata, fragment-pair information, and evaluation protocols related to the Fate Twin framework.

## Repository status

This repository has been created to support transparency and reproducibility during manuscript submission. The full code, derived metadata, and evaluation scripts will be progressively organised and released here as permitted by third-party data licences and project constraints.

## Data availability

The third-party manuscript images used in this study are subject to the licence restrictions of the original data providers and cannot be redistributed by the authors.

Therefore, this repository will not contain the original manuscript images. Where permitted, it will provide:

* derived simulation metadata;
* fragment-pair lists and label information;
* evaluation protocols;
* training and inference scripts;
* configuration files;
* instructions for reproducing the reported experiments using legally obtained source data.

## Planned structure

* `code/`: implementation of the Fate Twin simulation, visual realisation, and reassembly evaluation pipeline;
* `metadata/`: derived simulation metadata and fragment-pair information;
* `evaluation/`: scripts and protocols for reproducing the reported metrics;
* `data/`: documentation of data sources and data-use restrictions.
